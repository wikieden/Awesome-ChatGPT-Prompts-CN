## 0. 先读：这份指南解决什么问题

很多 AI 工具（ChatGPT、Claude 官方 App、Codex 等）只在**美区 App Store** 上架，且访问需要稳定的国际网络。核心链路是：

1. **科学上网**：装好代理客户端，关键场景（AI 工具、CLI 编程工具、注册美区 ID）开 **TUN 全局模式**；
2. **美区 Apple ID**：浏览器注册、地址填**免税州**、付款方式选 **None**；
3. **Apple 礼品卡（美区）**充值，在 **iOS App 内**用余额订阅扣款。

> ⚠️ **免责与合规提示**：本文仅作信息整理，请遵守当地法律法规与 Apple / OpenAI / Anthropic 的服务条款。使用真实可控的邮箱与手机号、开启两步验证；礼品卡务必走**正规渠道**，远离"超低价代充"（多为盗刷黑卡，会连累封号）。涉及付费的最终操作请你本人确认后执行。

### 核心概念：系统代理 vs TUN 模式（最重要）

代理客户端通常有两个开关，务必搞清区别：

- **系统代理（System Proxy / HTTP 代理）**：只对"会读取系统代理设置"的程序生效，主要是浏览器。**很多命令行工具、Electron 桌面应用、部分 AI 客户端不读系统代理**，于是会"连不上"或"走了真实 IP 泄露"。
- **TUN 模式（虚拟网卡 / 增强模式）**：创建一块虚拟网卡，在**网络层接管全部流量**。无论程序读不读代理设置——浏览器、桌面 App、`git` / `npm` / `curl`、**Codex CLI、Claude Code 等命令行编程工具**——统统被路由出去。

> ✅ **结论：跑 AI 工具、尤其是 CLI 编程工具（Codex CLI、Claude Code、gemini-cli 等），必须开 TUN 模式。** 只开系统代理，这些命令行工具大概率连不上或超时。

### Clash Verge Rev 使用方法（Win / Mac / Linux）

1. **安装**：到 [Clash Verge Rev GitHub Releases](https://github.com/clash-verge-rev/clash-verge-rev/releases) 下载对应系统安装包（Windows `.exe` / macOS `.dmg`，Apple 芯片选 arm64 / Linux `.deb`/`.AppImage`）。
2. **导入订阅**：「订阅(Profiles)」→ 粘贴机场给的 **Clash 订阅链接** → 导入 → 更新。
3. **选节点**：「代理(Proxies)」里选一个**美国节点**（注册美区 ID / 用美区服务时务必选美国）。
4. **开 TUN**：「设置」里打开 **TUN 模式（虚拟网卡）**。首次开启会要求**安装系统服务 / 授权管理员**，按提示装一次即可。
5. **模式**：日常用「规则(Rule)」模式即可；要"全部流量都走美国"（如注册 Apple ID）就切「全局(Global)」模式。
6. 需要时同时打开「系统代理」开关（浏览器用），但**TUN 才是覆盖 CLI 工具的关键**。

### FLClash 使用方法（Android）

1. 到 [FLClash GitHub Releases](https://github.com/chen08209/FlClash/releases) 下载 `FlClash-xxx-android-arm64-v8a.apk` 安装。
2. 「订阅」→ 添加 → 粘贴 Clash 订阅链接 → 更新。
3. 选美国节点 → 首页点启动，授予 **VPN 权限**（安卓上即等价 TUN，全局接管）。
4. 手机上的 ChatGPT / Claude App 直接就走代理了。

### Shadowrocket 使用方法（iOS）

1. 用**美区 Apple ID** 在 App Store 搜 **Shadowrocket** 购买下载（约 $2.99，用礼品卡余额即可付）。
2. 打开 → 右上角「+」→ 类型选 **Subscribe(订阅)** → 粘贴机场订阅链接 → 保存。
3. 选美国节点 → 首页开关打开 → 允许添加 **VPN 配置**（iOS 上即全局 TUN）。
4. 顶部"全局路由"可设为「代理(Proxy)」让全部流量走美国。

### ⭐ 注册美区 Apple ID 前：务必确认 IP 是美国

注册 / 激活美区 ID 时，**先开 TUN 全局代理、选美国节点**，然后**确认出口 IP 确实在美国**再动手——IP 不对是被风控、出不来 None 的常见原因。

打开下面任一链接确认（看 Country = United States、ISP 不是国内运营商）：

- **https://whoer.net/zh** —— 中文，显示 IP / 国家 / 是否被识别为代理
- **https://ipinfo.io/** —— 简洁，显示 IP 归属国家、城市、运营商
- **https://ip.sb/** —— 极简，直接看 IP 与国家

> 确认显示 **United States** 后再去注册。注册全程保持 TUN 全局、不要中途切节点/断线。激活成功后日常使用就不必一直全局了。

## 2. 美区 App Store 账号（Apple ID）

> 参考并实测整理（含防封要点）。下面是最稳的"全新注册"流程；已有账号改区的注意事项见文末小节。

### 黄金原则（先记住）

- **只在浏览器里注册，不要用 iPhone「设置」里的入口**——设置入口会绑定当前中国大陆地区，之后改区麻烦且易触发风控。
- **国家/地区选 United States，一旦创建不可改**，选定后永久保持。
- **美区号只登 App Store，永远不要登 iCloud**（你的照片/通讯录/备忘录都在国区 iCloud，互不影响才安全）。

### 第一步：官网创建账号（约 3 分钟）

浏览器打开 <https://account.apple.com>（或 appleid.apple.com）→「创建你的 Apple 账户」，关键字段一个都别错：

- **国家和地区：United States**（核心，创建后不能改）。
- **出生日期：填一个成年人**（建议 1990 年前后）。美区对 ChatGPT、Claude 等多标记 **17+**，若系统判定你未成年，这些 App **直接搜不到**，且极难改。
- **邮箱**：用**没注册过 Apple ID 的新邮箱**（QQ/163/Gmail 均可）；**不要用你国区 iCloud 那个邮箱**，避免关联风控。
- **手机号：建议填你自己的国内 +86 真实号**。Apple 官方允许大陆用户用 +86 验证；用自己的号便于日后**找回密码**。（一个号最多绑 3–5 个 Apple ID，超限就借家人号收个码，激活后可改绑；实在没有也可用接码平台 [hero-sms](https://hero-sms.com/cn) 过渡，但找回会更麻烦。）

### 第二步：在 App Store 激活（最容易翻车的一步）

1. iPhone 打开 **App Store** → 右上角头像 → 滑到底 → **退出登录**。
   > ⚠️ **只退 App Store，永远不要退 iCloud！** 两者只差一屏，但退错 iCloud 会影响你的全部个人数据。
2. **触发 None 的隐藏技巧**：不要直接点头像登录，而是**直接在 App Store 搜 ChatGPT（或任意免费 App）→ 点「获取(Get)」**→ 弹出登录时输入新美区账号密码 → 顺流程走到付款方式页 → **「None」几乎必然出现** → 选 **None**。
3. **填美国地址**（强烈建议免税州，见下）。
4. **Review 隐藏关卡（正常，别慌）**：首次下载时可能弹"此 Apple ID 尚未在 iTunes Store 使用过 → 检查(Review)"，这是标准验证：打开"同意条款"开关 → 确认地址 → 完成。通过这一次以后就不再弹。

> 极少数情况下（约 5%）反复换网络 None 仍不出现——这通常是当前网络被打了地区标签。**此时务必开 TUN 全局美国节点、确认 IP 为美国（见第 1 节）再走激活**，激活成功后即可恢复直连。

### 免税州地址（省销售税的关键）

美国按账单地址收销售税，全美只有 **5 个州没有州销售税**。**这 5 个免税州是**：

1. **Oregon（俄勒冈）** —— 常用城市 Portland，邮编示例 97204
2. **Delaware（特拉华）** —— 常用城市 Wilmington，邮编示例 19801
3. **Montana（蒙大拿）** —— 常用城市 Helena，邮编示例 59601
4. **New Hampshire（新罕布什尔）** —— 常用城市 Concord，邮编示例 03301
5. **Alaska（阿拉斯加）** —— 常用城市 Anchorage，邮编示例 99501

> 记忆口诀：**俄勒冈、特拉华、蒙大拿、新罕布什尔、阿拉斯加**。其中 Oregon、Delaware 最常用。

**生成可用地址**（州 / 城市 / 邮编必须互相匹配）：

- **免税州地址生成器（推荐，直接出免税州地址）**：<https://addressgen.top/zh/us-tax-free-address-generator> —— 中文界面，专出上述免税州的合规地址。
- **美国通用地址生成器**：<https://mockaddress.com/usa-address/> —— 生成街道/城市/州/ZIP/电话，记得把**州手动选成免税州**（如 Oregon / Delaware）。
- **更稳妥的做法**：用 **Google 地图**搜波特兰(Portland)等免税州城市里的麦当劳/星巴克/商场，把真实门牌号+ZIP 抄下来填（门牌号、电话尾数可微调）。Apple 不核验你是否真住那里，但真实存在的地址更不易被标记。

> 填写要点：**州 = 免税州**、**ZIP 与州匹配**、城市真实。州和邮编对不上是最常见失败原因。

### 养号：新号头 3–5 天别急着花钱

刚注册的跨区新号 = "0 天新号 + 跨区标签"，**立刻充值订阅几乎必被拦**（弹"购买无法完成，请联系 iTunes 支持"，余额可能被冻）。正确做法：

1. 注册激活后**先别充值**，在美区 App Store **下载 3–4 个免费 App**（ChatGPT 免费版、YouTube、Spotify 等）——下载行为会被记为正常使用。
2. 让这些 App 在手机上**正常存在 1–2 天**，隔天打开用一下更好。
3. **至少等 3 天（5 天更稳）再充值**。
4. **首次充最小面额（$5 / $10）**，先买一个便宜 App 或 $0.99 内购，留下一条"正常消费"记录；之后再大额充值就不易被拦。

### 已有账号改区到美区（备选）

不想另建号也可改区，**改区前必须**：① 取消所有订阅；② 账户余额清零到 $0；③ 无待结算项。然后：设置 → 媒体与购买项目 → 查看账户 → 国家/地区 → United States → 付款方式 None → 填免税州地址。
> ⚠️ 改区会让原区独占的已购/订阅失效，且不能频繁来回切。重度依赖原区的人建议用上面的"全新注册"。

## 3. Apple 礼品卡购买与充值

养号期过后再充值。礼品卡充进美区 ID，之后订阅都从**余额**扣，免绑卡。

### 必须是美区美元卡

其它国家的 Apple Gift Card **不能跨区使用**。常见面额 $5 / $10 / $25 / $50 / $100，**首次只买小额（$5 / $10）**。

### 购买渠道

- **Pockyt 官方商城（国内最方便，推荐）**：浏览器直接打开 **<https://shop.pockyt.io/pc/home>** → 选「App Store & iTunes（美国 / US）」→ 选小面额（首次 $5 / $10）→ **支持国内银行卡、微信、支付宝**付款 → 即时拿到兑换码。无需切定位、无需小程序，最省事。
- **支付宝小程序（备选）**：支付宝左上角定位切到美国城市（如旧金山）→ 搜"Pockyt Shop / 出境"小程序 → 选「App Store & iTunes」→ 选小面额付款 → 拿到兑换码。（汇率约比官方高 5%。）
- **Apple 美国官网直购（永久备用）**：浏览器开 <https://www.apple.com/shop/gift-cards>（**美国官网**）→ 选 Email 类型 Gift Card → 收件邮箱填 QQ 即可、账单地址用免税州地址 → **结账选 Guest Checkout 访客身份、别登录 Apple ID** → 国内双币 Visa/Master 付款 → 兑换码发到邮箱。
- 也可在信誉好的卡商处买（认准好评、先验后付）。

### 兑换到账户

iPhone 登录**美区 ID** → **App Store → 头像 → 兑换充值卡或代码（Redeem Gift Card or Code）** → 输入/扫码兑换 → 余额即时到账。（或网页 <https://apps.apple.com/redeem>。）

### 避坑

- 卡区服必须 = Apple ID 区服（**美卡配美区号**）；
- **别碰"明显低于面值"的卡**（盗刷黑卡，入账后可能被清零并连累封号）；
- 余额不退、不可提现、不能转其他 ID/跨区，按需小额多次充。

## 4. Claude 账号注册与订阅

### 注册（务必美国 IP + 美国号 + 无痕）

> ⚠️ **和 Apple ID 不同**：Apple ID 绑**自己的中国手机号**就行；但 **Claude 和 OpenAI 注册必须用美国手机号**，且**全程美国 IP、浏览器开无痕模式**，否则极易触发风控被封号。

1. **开 TUN 全局 + 美国节点**，先到 [whoer.net](https://whoer.net/zh) 确认 IP = United States（全程不要断线、不要切节点）；
2. 浏览器开**无痕 / 隐私窗口**（Chrome：⌘/Ctrl+Shift+N；Safari：⌘+Shift+N）——避免带入旧 Cookie / 历史身份；
3. 打开 <https://claude.ai> →「Sign up」→ 用**邮箱**或 **Google** 注册 → 收邮箱验证码；
4. 若要求手机验证：用**美国号码**（接码平台 [hero-sms](https://hero-sms.com/cn) 选 United States 的号收码）；
5. 完成即 **Free**。

### 订阅档位（2026 年）

| 档位 | 价格（美元/月） | 适合 |
|---|---|---|
| Free | $0 | 轻度尝鲜 |
| **Pro** | **$20** | 日常重度、Claude Code 入门 |
| **Max 5×** | **$100** | 高频、长任务、Claude Code 重度 |
| **Max 20×** | **$200** | 跑 Agent / 持续高负载，按量更划算 |

### 用礼品卡付费（走 iOS 内购）

1. iPhone 登录**美区 ID** 且**有礼品卡余额**；
2. App Store 装 **Claude** by Anthropic；
3. App 内登录你的 Claude 账号 → Upgrade → 选 **Pro / Max** → 走 **Apple 内购**，从余额扣款；
4. 会员在**网页版 claude.ai 同账号**通用（同邮箱登录即可）。

> claude.ai **网页订阅走信用卡，用不了礼品卡**；要用礼品卡就必须从 **iOS App 内**订阅。iOS 内购价含 Apple 抽成、略高属正常。
>
> 用 **Claude Code（CLI）** 时：本机需开 **TUN 模式**，否则命令行连不上（见第 1 节）。

## 5. OpenAI ChatGPT 账号注册 + Codex 激活

> ⚠️ OpenAI 风控很严：**必须美国手机号 + 全程美国 IP + 浏览器无痕模式**，中国号 / 国内 IP / 带旧 Cookie 极易注册失败或被封。

1. **开 TUN 全局 + 美国节点**，到 [whoer.net](https://whoer.net/zh) 确认 IP = United States（全程保持，别断别切）；
2. 浏览器开**无痕 / 隐私窗口**；
3. 打开 <https://chatgpt.com> →「Sign up」→ 邮箱 / Google 注册 → 邮箱验证码；
4. 手机验证用**美国号码**（[hero-sms](https://hero-sms.com/cn) 选 United States 取号收码）；
5. 完成即 **Free**。

**Codex 已包含在所有 ChatGPT 套餐里，没有独立的 Codex 订阅。** 订哪个 ChatGPT 档，就自带对应额度的 Codex。

> 用量打满后，Plus / Pro 可**单独购买额外 credits** 继续用，不必升级整档。

### 激活 / 使用 Codex

- **网页/桌面**：登录 ChatGPT 后在侧边栏进入 **Codex**（云端写代码智能体）；

- ⚠️ **Codex CLI、Claude Code 等命令行工具务必开 TUN 模式**——它们不走系统代理，只开系统代理会连不上。

### 用礼品卡付费（走 iOS 内购）

## 7. 常见问题 / 避坑速查

- **CLI 编程工具（Codex/Claude Code）连不上？** 八成只开了系统代理——**改开 TUN 模式**。
- **注册美区 ID 出不来 None？** 先开 **TUN 全局美国节点**、用 [whoer.net](https://whoer.net/zh) 确认 IP 是美国，再走"搜 ChatGPT 点 Get"触发法。
- **新号充值被拦？** 没养号——先下免费 App 用几天，3–5 天后再小额充值。
- **网页订阅能用礼品卡吗？** 不能，礼品卡只走 iOS 内购。
- **iOS 内购比官网贵？** 含 Apple 抽成，略高正常。
- **改区后老 App 没了？** 跨区已购可能失效，重度用户单独建美区号。
- **手机号到底用中国号还是美国号？** **Apple ID** 用**自己的中国 +86 号**（便于找回）；**Claude / OpenAI** 必须用**美国号**，且全程美国 IP + 浏览器无痕，否则易被封。
- **Apple ID 想找回密码？** 用自己的国内号注册，找回靠它；接码号会让找回更难。
- **Shadowrocket 装不了？** 它要美区 ID——先用电脑 Clash Verge Rev 开 TUN 注册好美区 ID，再下载它。

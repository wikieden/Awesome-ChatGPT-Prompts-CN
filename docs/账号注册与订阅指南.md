# 美区账号注册与订阅指南（App Store / Claude / OpenAI Codex）

> 面向新手：如何注册一个可用的**美区 App Store 账号**，注册 **Claude** 与 **OpenAI ChatGPT / Codex** 账号，并用 **Apple 礼品卡**完成订阅付费（无需绑定国外信用卡）。

## 0. 先读：这份指南解决什么问题

很多 AI 工具（ChatGPT、Claude 的手机端订阅、部分应用）只在**美区 App Store** 上架，或在美区价格/可用性更友好。核心思路是：

1. 有一个**纯美区的 Apple ID**（地址填**免税州**，省掉销售税）；
2. 用 **Apple 礼品卡（美区）**给这个 Apple ID 充值；
3. 在 **iOS App 内**用 Apple ID 余额完成订阅扣款（这样就不需要国外信用卡 / PayPal）。

> ⚠️ **免责与合规提示**：本文仅作信息整理。注册海外账号、跨区订阅需遵守 Apple、OpenAI、Anthropic 各自的服务条款；请使用真实可控的邮箱，妥善保管密保与两步验证。礼品卡务必从**正规渠道**购买，警惕"低价代充/代付"骗局与盗刷卡。涉及付费的最终操作（充值、订阅）请你本人确认后执行。

## 1. 美区 App Store 账号（Apple ID）

有两条路径，按你的情况二选一。

### 方式 A：全新注册一个纯美区 Apple ID（推荐，最干净）

1. **准备一个干净邮箱**：建议用 Gmail / Outlook 新邮箱，不要用已经绑过其他区 Apple ID 的邮箱。
2. **浏览器注册**：打开 <https://account.apple.com> →「创建您的 Apple 账户」。
   - 出生日期需满足年龄要求；
   - **国家/地区**这一步如果能选就选 **United States**（部分入口在注册时不让选区，没关系，付款方式那步改即可）。
3. **在 iPhone 上登录并切换到美区**：
   - 「设置 → 顶部你的名字 → 媒体与购买项目 → 查看账户 → 国家/地区 → 更改国家或地区 → United States」。
   - **付款方式选「无 / None」**（关键！选 None 才能跳过信用卡）。
4. **填美区地址**（见下方「免税州地址来源」）：街道、城市、州、ZIP 邮编、电话（美国号码格式，可用接码平台 [hero-sms](https://hero-sms.com/cn) 拿一个美国号验证）。
5. 保存。此时你就有了一个**付款方式为 None 的美区 Apple ID**，接着用礼品卡充值即可（见第 2 节）。

> 小贴士：能选 None 的前提通常是账户**没有未付清的订阅/欠款**、且地区信息一致。如果 None 选项不出现，先用一张美区礼品卡充值（第 2 节）往往就能跳过绑卡。

### 方式 B：把现有 Apple ID 改区到美区

适合不想再记一套账号的人。**改区前必须做完三件事**，否则会卡住：

1. **取消所有订阅**：设置 → 你的名字 → 订阅 → 全部取消，等到期。
2. **花光账户余额**：余额必须为 **$0 / ¥0** 才能改区（买点小东西或等清零）。
3. **退出 iCloud 相关结算项**：确认没有待结算的家庭共享/App 内购。

然后：设置 → 媒体与购买项目 → 查看账户 → 国家/地区 → 改为 **United States** → 付款方式选 **None** → 填免税州地址 → 保存。

> ⚠️ 改区会让你**原区已购但未在美区上架的 App / 订阅失效**，且不能频繁来回切换。重度依赖原区的人建议用方式 A 单独建号。

### 免税州地址来源（省销售税的关键）

美国按账单地址收销售税，**这 5 个州没有州销售税**，把 Apple ID 地址填成它们能省掉订阅税费：

| 免税州 | 常用城市 | 邮编示例 |
|---|---|---|
| **Oregon（俄勒冈）** | Portland | 97086 |
| **Delaware（特拉华）** | Newark | 19711 |
| **Montana（蒙大拿）** | Helena | 59601 |
| **New Hampshire（新罕布什尔）** | Concord | 03301 |
| **Alaska（阿拉斯加）** | 多数地区无州税 | 99501 |

**怎么找一个能通过验证的真实地址**（不要随便瞎填，建议用真实存在的地址格式）：

- 用 **Google 地图**搜上述城市里的**商场、便利店、酒店**，复制它的街道地址 + ZIP；
- 或用"美国地址生成器"类网站生成符合该州格式的地址；
- 电话填美国格式（区号 + 7 位），需要短信验证时用接码平台 [hero-sms](https://hero-sms.com/cn) 取一个**美国号码**。

> 填写要点：**州 = 免税州**、**ZIP 与州匹配**、城市真实。州和邮编对不上是最常见的失败原因。

## 2. Apple 礼品卡购买与充值

用礼品卡给美区 Apple ID 充值，之后所有订阅都从**账户余额**扣，免绑卡。

### 买哪种、去哪买

- 必须是 **美区 Apple Gift Card（美元面额）**——其它国家的 App Store / Apple Gift Card **不能跨区使用**。
- **正规渠道**：Apple 官网美国站、Amazon US、PayPal Digital Gifts、Target/Walmart 等线下零售；国内可在**信誉好的代购/卡商**处买（认准好评、支持先验后付）。
- 常见面额 $10 / $15 / $25 / $50 / $100，按你的订阅需求买（订阅是按月扣，建议先买够 1–2 个月）。

### 如何兑换到账户

**手机端（最稳）**：
1. 确认 iPhone 已登录**美区 Apple ID**；
2. 打开 **App Store** → 右上角头像 → **兑换充值卡或代码（Redeem Gift Card or Code）**；
3. 输入卡背面的兑换码（可用"使用相机"扫码）→ 余额入账。

**也可以**：打开链接 <https://apps.apple.com/redeem> 登录后兑换。

### 避坑

- 卡码区服必须 = Apple ID 区服（**美卡配美区号**）；
- 别买"明显低于面值"的卡，多为盗刷卡，入账后可能被 Apple 冻结；
- 兑换后在 App Store 头像页能看到余额（Account → 余额）。

## 3. Claude 账号注册与订阅

### 注册

1. 打开 <https://claude.ai> →「Sign up」；
2. 可用 **邮箱**或 **Google 账号**注册，邮箱会收验证码；
3. 完成后即为 **Free（免费版）**，可日常体验。

### 订阅档位（2026 年）

| 档位 | 价格（美元/月） | 适合 |
|---|---|---|
| Free | $0 | 轻度尝鲜 |
| **Pro** | **$20** | 日常重度使用、Claude Code 入门 |
| **Max 5×** | **$100** | 高频、长任务、Claude Code 重度 |
| **Max 20×** | **$200** | 跑 Agent / 持续高负载，按量更划算 |

### 用 Apple 礼品卡付费的正确路径

礼品卡只能通过 **iOS App 内购**扣款，所以：

1. iPhone 登录**美区 Apple ID**且**已有礼品卡余额**（第 2 节）；
2. App Store 搜 **Claude** by Anthropic → 安装；
3. App 内用你的 Claude 账号登录 → 进入升级 / Upgrade → 选 **Pro / Max** → 走 **Apple 内购**结算，自动从 Apple ID 余额扣款；
4. 订阅成功后，这个会员在**网页版 claude.ai 同账号**也生效（同一邮箱登录即可）。

> 注意：在 **claude.ai 网页**直接订阅走的是信用卡通道，**不能用礼品卡**。要用礼品卡就必须从 **iOS App 内**订阅。iOS 内购价可能比官网略高（含 Apple 抽成），属正常。

## 4. OpenAI ChatGPT 账号注册 + Codex 激活

### 注册

1. 打开 <https://chatgpt.com> →「Sign up」；
2. 邮箱 / Google / Apple / 微软 账号均可，邮箱需验证；
3. 注册时可能需要**手机号验证**——用接码平台 [hero-sms](https://hero-sms.com/cn) 取号收码；
4. 完成后即 **Free**。

### Codex 怎么来：它不是单独订阅

**Codex 已包含在所有 ChatGPT 套餐里，没有独立的 Codex 订阅。** 你订哪个 ChatGPT 档，就自带对应额度的 Codex（用量随档位提升）。

| ChatGPT 档位（2026） | 价格（美元/月） | Codex 用量 / 说明 |
|---|---|---|
| Free | $0 | 基础体验 |
| Go | $8 | 入门加量 |
| **Plus** | **$20** | 日常 + 基础 Codex 用量 |
| **Pro（Codex）** | **$100** | 比 Plus 多约 5× Codex 用量，含全部 Pro 功能（2026-04-09 起新增） |
| **Pro（Max）** | **$200** | 最高用量 / 20× 档 |

> 用量打满后，Plus / Pro 可**单独购买额外 credits** 继续用，不必升级整档。

### 激活 / 使用 Codex

- **在 ChatGPT 里**：登录后在侧边栏 / 工具区进入 **Codex**（云端智能体写代码）；
- **在终端**：安装 Codex CLI（`npm i -g @openai/codex` 之类，按官方文档为准），用同一 OpenAI 账号登录授权即可；
- 订阅生效后 Codex 额度自动可用，无需另外"激活码"。

### 用 Apple 礼品卡付费的路径

和 Claude 同理，礼品卡走 **iOS 内购**：

1. iPhone 登录**美区 Apple ID** + 有礼品卡余额；
2. App Store 装 **ChatGPT** by OpenAI；
3. App 内登录你的 OpenAI 账号 → Upgrade → 选 **Plus / Pro** → **Apple 内购**结算，从余额扣款；
4. 会员在**网页版 chatgpt.com 同账号**通用。

> 同样地，chatgpt.com 网页订阅走信用卡，**要用礼品卡必须从 iOS App 内订阅**。

## 5. 通用原理一句话总结

> **美区 Apple ID（免税州地址）→ 充美区 Apple 礼品卡 → 在 iOS App 内订阅 → 从 Apple 余额扣款，网页同账号通用。**

这条链路的好处：不用国外信用卡、地址免税、扣款可控（余额花完自动停，不会被持续扣费）。

## 6. 常见问题 / 避坑速查

- **能选 None 跳过绑卡吗？** 账户无欠款 + 地区一致时通常可以；不行就先充一张礼品卡再试。
- **网页订阅能用礼品卡吗？** 不能，礼品卡只走 iOS 内购。
- **iOS 内购比官网贵？** 含 Apple 抽成，略高属正常，换来的是可用礼品卡付费。
- **改区后老 App 没了？** 跨区订阅 / 已购可能失效，重度用户建议单独建美区号（方式 A）。
- **手机号验证收不到码？** 用 [hero-sms](https://hero-sms.com/cn) 选**对应国家**的号码，注意区服一致。
- **要科学上网吗？** 访问及部分注册步骤需要稳定的国际网络，机场推荐见 [README](../README.md) 顶部。
